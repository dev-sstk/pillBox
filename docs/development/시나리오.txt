# 처음 시작하는 유저를 위한 안내
전원 ON -> 필박스 Startup Screen 출력 -> "필박스 초기 설정을 시작합니다" 텍스트와 음성 출력 -> 와이파이 선택 화면 및 스캔 및 연결(비밀번호 입력 시 키패드 필요), 백그라운드에서 타임서버와 시간 동기화 -> 약 복용 횟수 선택 1, 2, 3 중 선택 -> 복용 시간 선택 00:00 (시간, 분 단위), 복용 횟수가 2회면 복용시간을 2개 저장할 수 있게 함 -> 복용 일 수 선택(최대 저장 디스크 한개당 10개라고 하면, 3개면 30일 선택 가능) -> "알약을 채워주세요" 텍스트와 음성 출력 후, "1번 약을 넣어주세요" 텍스트 출력, 채운 후, 선택버튼을 눌러 다음으로 스텝모터 3칸 회전(리미트스위치로 칸 수 세기), 다 채웠으면 2번 디스크, 3번 디스크 순으로 채운 후, 다 채웠으면 "알약 리필을 완료했습니다" 텍스트와 음성 출력 후 홈화면으로 전환
홈 화면에서 볼 수 있는 것들 - 와이파이 연결상태 및 신호세기, 배터리 잔량, 현재 시간, 복용 예약 시간, 복용 일수, 복용 상태 체크, 저장 디스크 마다의 남은 알약 갯수

약 복용 시간이 되면 - "약 드실 시간이에요" 텍스트와 음성 출력 1초 간격으로 울림, 10분 간 반복 후 복용하지 않으면 알림 종료 및 복용 상태에 미복용으로 체크, 복용하려면 선택 버튼을 눌러야 함, 선택 버튼 누르면 "알약 배출" 텍스트와 음성출력 후 알약 배출

약을 다 먹었으면 - "알약을 채워주세요" 텍스트와 음성 출력, 선택버튼을 누르면 초기 설정에서 "약 복용 횟수 선택"부터 설정 시작

config 파일에 저장할 값
스텝모터 위치, 와이파이 정보(SSID, 비밀번호) 전원 껐다 켰을 때 신호가 있으면 자동 연결, 타임서버주소, 복용 횟수, 복용 시간, 남은 알약 수, 복용 상태 체크

1. 전원 ON 및 초기화
스텝모터 위치, 현재 모드 config에서 읽기
메뉴 홈 화면 출력

2. 알약 리필
- 리필 모드 시작
선택버튼 3초간 누름 - (복용 예약이 되어 있지 않으면 먼저 예약하라는 안내음성 출력 "복용 예약이 되어 있지 않습니다.", "복용 예약 메뉴로 이동합니다", "이동하시겠습니까?" 음성출력 후 화면에 예/아니오 선택하도록 출력) - "리필 모드" 음성 출력되면서 화면에 "리필 모드" 텍스트 출력 후 다시 홈화면으로 전환
스텝모터 1, 2, 3 원점 이동
스텝모터 위치, 현재 모드 config에 저장

- 다음 칸 이동
선택버튼 누름 - 스텝모터 1, 2, 3 다음 세 칸 이동 - 공급칸이 총 몇개지?
스텝모터 위치, 현재 모드 config에 저장

선택버튼 3초간 누름 - "리필 종료" 음성 출력
스텝모터 위치, 현재 모드 config에 저장

3. 복용 예약
- 하루 몇 회: 1~3회 선택
- 시간 입력: 몇 시 몇 분 / 형식 00:00 / 예) 12시 00분 -> 12:00 / 입력할 부분은 화면에서 깜빡임, 위/아래 버튼으로 시 단위 값 더하고 빼기, 그 상태에서 선택 버튼 누르면 분 단위 입력 깜빡임, 마찬가지로 입력 후 선택 버튼 눌러서 저장, 하루 복용 횟수에 따라서 시간 입력 칸 갯수 적용함
- 전부 입력하고 나오려면 메뉴 버튼 누름
- 나오기 전에 저장하시겠습니까? 예/아니오 선택하도록 출력
